# 泽小白工具链
包含工具：

1. Lable2Rec：  
`Audacity标签轨转录音表工具`    
`用法：Lable2Rec <rec> <lable> [sflg]`  
`rec：输出录音表`  
`lable：标签轨`  
`sflg：跳过的标签（通常是第一个 ）`  
`省略sflg将跳过空标签`
2. Dir2Rec  
`基于文件列表的录音表构建工具`  
`用法：Dir2Rec <rec> [dir]`  
`rec：输出录音表`  
`dir：文件夹`  
3. WaveMove  
`音频文件整理工具（按标签整理音频文件）`  
`用法：WaveMove <lable> <dir> [sflg]`  
`lable：标签轨`  
`dir：移动到的文件夹`  
`sflg：跳过的标签（通常是第一个 ）`  
`省略sflg将跳过空标签`
